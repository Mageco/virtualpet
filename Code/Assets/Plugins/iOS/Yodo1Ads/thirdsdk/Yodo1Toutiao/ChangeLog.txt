2020-02-3
1.BUAdSDK v2.7.5.2 

注意事项（必看）：
1. 从2752版本开始，只支持x86_64, armv7, armv7s, arm64这几种架构，不再支持i386.
2. 为了优化zip包的下载速度，demo中只放置了Podfile而没有运行，请先cd到BUDemo文件夹下，运行pod install，成功后，即可正常运行demo。
3. 2752版本在pod install 之前请先安装Git-LFS服务，安装方法：https://github.com/bytedance/Bytedance-UnionAD；如果安装后仍不行，请先清除pod的本地缓存（pod cache clean Bytedance-UnionAD），再重新pod install。

优化点：
1. 支持模板开屏广告；
2. 优化了开屏广告的加载速度，优化开屏广告的回调准确性；
3. playable广告优化；
4. 解决模拟器偶现崩溃问题；
5. 解决了模板广告WKWebview的body拦截问题；
